> Cynara 是一个非营利性质的工作室，因此目前无法提供常规意义上的劳动报酬。

# Let's Shine Brighter Together.

Cynara 希望持续创作有内容、有表达、也有长期价值的作品。如果你认同这样的方向，欢迎了解我们的岗位需求。

## 空缺岗位

### 法务

#### 职位描述

1. 根据不同业务线的需求，协助检索、整合法律法规、政策、案例与平台规则，并给出风险提示或处理建议。
2. 协助审核、起草与归档部分法律文件，确保资料可追踪、可检索。
3. 协助处理公司相关诉讼与法务事务。

#### 职位要求

1. 具备扎实的法律理论基础，并对文娱法、网络法、知识产权法有一定了解。
2. 了解企业法务常见工作流程，并对诉讼实务有基础认知。
3. 熟悉常规办公软件与法律工具，具备较强的文书处理与检索能力。
4. 工作认真细致、责任感强，具备独立思考与持续学习能力。

### 美术设计

#### 职位描述

1. 负责公司软件、网站、移动端应用等产品的视觉与界面设计。
2. 建立并维护统一的设计规范，保持品牌视觉一致性。
3. 关注行业趋势，尝试将新的视觉元素与创意方向融入产品设计。

#### 职位要求

1. 熟练掌握 Photoshop、Illustrator 等主流设计工具，具备 UI 与动效设计能力。
2. 有较好的审美判断力，能把控色彩、构图与整体氛围。
3. 吃苦耐劳，责任感强，能与团队保持协作。
4. 学习能力强，愿意持续吸收新的设计方法与行业变化。

### 视频账号运营

#### 职位描述

1. 独立负责视频账号内容，保证内容调性与整体质量符合要求。
2. 能够主动提出二创思路，并参与内容节奏与表达方式的优化。

#### 职位要求

1. 具备良好审美，对影视、自媒体内容与 BGM 选择有较强感知。
2. 具有节奏把控能力、网感与一定幽默感。
3. 吃苦耐劳，责任感强，能与团队长期配合。

### 后端工程师

#### 职位描述

1. 使用 Node.js、Python 等语言开发公司产品的后端功能模块。
2. 将业务需求转化为稳定、可维护的服务逻辑，并持续优化性能。
3. 全程参与项目，从需求沟通、开发测试到上线维护。

#### 职位要求

1. 熟悉 Node.js、Python 及其常见生态，能够独立承担复杂模块开发。
2. 熟悉 GitHub 协作流程，包括版本管理、代码提交与冲突处理。
3. 具备良好的团队协作能力与主动沟通意识。
4. 学习能力强，愿意持续关注行业技术变化。
5. 有趣、靠谱，能在长期协作中保持稳定输出。

### 前端工程师

#### 职位描述

1. 使用 Node.js、HTML、CSS、JavaScript 等技术开发公司产品的前端界面。
2. 将业务需求转化为高质量交互与页面结构，并兼顾性能与维护性。
3. 深度参与项目全流程，与团队一起推进产品上线与迭代。

#### 职位要求

1. 熟悉 Node.js、HTML、CSS、JavaScript，并能独立完成前端模块开发。
2. 熟悉 GitHub 协作流程，包括版本管理、分支协作与冲突处理。
3. 具备良好的团队沟通能力与执行力。
4. 学习能力强，愿意主动关注技术趋势。
5. 有责任感，也有自己的表达与想法。
