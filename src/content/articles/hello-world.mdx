### 为什么选择 MDX？
在标准的 Markdown 中，您通常只能编写结构化文本。但是借助于 [MDX](https://mdxjs.com/)，我们可以在静态的内容里直接嵌入 **可交互的框架组件**。这也就意味着，您可以一边写着优雅的技术文档，一边向用户展现酷炫的前端 Demo。

```javascript
// 代码高亮也被原生支持了
function WeCreateTheFuture() {
  console.log("Future is here.");
}
WeCreateTheFuture();
```
